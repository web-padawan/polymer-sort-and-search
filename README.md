#Polymer Sort and Search
This is a demo project implementing [React Challenge #1](http://jsraccoon.ru/react-challenge-sort-and-search) task with [Polymer](https://www.polymer-project.org/).

## Installation

* npm i -g gulp
* npm i
* gulp
* gulp serve
